Proposito del sitio

Este recurso educativo digital tiene como objetivo principal, mejorar las estrategias de aprendizaje, por medio de la enseñanza interactiva, comprendiendo las etiquetas basicas de lenguaje HTML, atributos y valores, para posteriormente desarrollar su respectiva aplicación. HTML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado HTML) para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros…

Atributos

Son elementos de las etiquetas que pueden proporcionar ciertas caracteristicas a un texto, imagen etc. Se insertan dentro de la etiqueta de inicio y llevan detras el signo = con dicho valor del atributo que se quiere aplicar.
Se clasifican generalmente en requeridos, opcionales, estándares (también denominados globales) y de evento. Normalmente los atributos requeridos y opcionales modifican elementos de HTML concretos, mientras que los atributos estándares pueden aplicarse a la mayoría de elementos de HTML. Los atributos de evento, añadidos en versión HTML 4, permiten que un elemento especifiquen guiones o script que se ejecutaran en ciertas circunstancias

Valores

Afectan directamente al elemento, pueden estar encerrados entre comillas simples o dobles, dejar valores sin comillas se considera inseguro, en palabras mas sencillas es la acción que va a realizar el atributo.
